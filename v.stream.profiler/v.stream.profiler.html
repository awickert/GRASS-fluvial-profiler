DESCRIPTION

v.stream.profiler builds and exports plots and data sets of river long profiles, slope-accumulation (e.g., slope-area), and more for topographic analysis. Starting from a chosen segment (cat), it walks either downstream to the map outlet (a single channel) or upstream through every tributary that drains to it. Elevation, flow accumulation, and slope rasters are sampled along the channel; the profile can be densified to a target spacing (dx_target) and smoothed with a moving-average window (window). Results can be plotted, written as a text table (outfile), and exported as a NetworkX node-link JSON sub-network (json).

NOTES

v.stream.profiler requires its input streams vector to have been linked by v.stream.network first (it reads the tostream column to walk the network); it will stop with an error otherwise. The graph construction, raster sampling, and JSON I/O are provided by the rivernetworkx Python package, which must be importable in the GRASS Python environment (pip install -e .).
