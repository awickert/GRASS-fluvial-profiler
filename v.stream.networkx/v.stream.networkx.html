DESCRIPTION

v.stream.networkx converts a river network built by r.stream.extract (and linked by v.stream.network) into a NetworkX directed graph and, optionally, exports it as a JSON file.

Each stream segment becomes a directed edge that points to its downstream neighbour; the segment that leaves the map drains to node 0, which represents the outlet. Geometry and raster values sampled along each segment are attached to the edges, and the values shared at tributary junctions are moved onto the nodes so that adjacent segments agree at their common point.

The following per-point arrays are computed along each segment and stored on the edges (and, at junctions, on the nodes):

  • x, y: map coordinates
  • s_upstream: along-segment distance measured from the downstream end
  • s_downstream: along-segment distance measured from the upstream end
  • z: elevation, if an elevation raster is given
  • A: flow accumulation, if an accumulation raster is given
  • s: cumulative distance upstream of the outlet, accumulated across the whole network by a breadth-first sweep from node 0

NOTES

streams must be a vector line network generated by r.stream.extract and first passed through v.stream.network, which adds the tostream column used here to build the graph topology.

The units and accum_mult options only set axis labels and scale the accumulation values; they do not change the network structure.

EXAMPLE

Build a stream network from a DEM and export it as NetworkX JSON (GRASS shell): 
# Region and per-cell area (for area-weighted flow accumulation).
# For a projected (metric) location the cell area is just nsres * ewres.
g.region -p raster=DEM
r.mapcalc "cellArea_meters2 = nsres() * ewres()" --overwrite

# Flow accumulation in m^2, masked to the DEM
r.watershed elevation=DEM flow=cellArea_meters2 \
    accumulation=accumulation -s --overwrite
r.mapcalc "accumulation = if(isnull(DEM), null(), accumulation)" --overwrite

# Extract the stream network
r.stream.extract elevation=DEM accumulation=accumulation \
    stream_raster=streams stream_vector=streams \
    threshold=30000000 direction=draindir d8cut=0 --overwrite

# Link each segment to its downstream neighbour, then build the graph
v.stream.network map=streams
v.stream.networkx streams=streams elevation=DEM accumulation=accumulation \
    units=m2 outjson=network.json
The exported JSON can be read back into a NetworkX graph and walked upstream from the outlet (node 0) to plot long profiles (Python): 
import json
import networkx as nx
from networkx.readwrite import json_graph
from matplotlib import pyplot as plt

with open("network.json") as f:
    G = json_graph.node_link_graph(json.load(f))

# Breadth-first sweep upstream from a starting node (the outlet is 0)
def bfs_upward(G, start):
    R = G.reverse(copy=False)  # view, no data copied
    for node in nx.bfs_tree(R, start):
        yield node

plt.figure()
for n in bfs_upward(G, 0):
    for parent, child in G.in_edges(n):
        e = G.edges[parent, child]
        plt.plot(e['s'], e['z'], 'k-', linewidth=2)
plt.xlabel('Distance upstream of outlet')
plt.ylabel('Elevation')
plt.show()
